# Support

Use public GitHub Issues for this adapter for:

- adapter configuration bugs,
- validation/doctor failures,
- documentation drift and onboarding friction.

For security reports, use a private advisory at the repository security page.
Do **not** open public issue text for exploit details.

When filing an issue, include:

- module OS and environment,
- exact command chain,
- relevant commit SHA,
- sanitized output from validation/installation commands,
- if available, minimal reproducible steps with repository state.

If this is a security issue, use a private advisory and avoid posting secrets,
keys, cookies, or endpoint credentials.
